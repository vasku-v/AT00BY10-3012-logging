# Project Documentation

This repository contains Vasilissa Vilkki's submission for the Logging
assignment in the course "AT00BY10-3012 Ohjelmistojen ylläpito ja testaus".

The repository contains two different tasks, and each task is located in its own
directory and includes its own documentation, source code and tests.

## Tasks

### Task 1 - Basic Tally Counter

A simple command-line based tally counter implementation.

**Location:**

- [TASK1](./task1/)

### Task 2 - Tally Counter REST API

A REST API built with Node.js, Express and Winston. Includes structured logging,
unit tests, integration tests and documentation.

**Location:**
